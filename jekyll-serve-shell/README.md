# DOC

## Clone

```
git clone https://github.com/rizkytegar/jekyll-server-sh
```

## Run 

CD to Your project

example :

```
cd /myproject
```

Run

```
./build.sh
```

Replace Gemspec with your Gemspec

find in cibuild.sh

My Gemspec

```
gem build cgsweb.gemspec
```

Example for you

```
gem build myweb.gemspec
```

and run 

```
./cibuild.sh
./server.sh
```
